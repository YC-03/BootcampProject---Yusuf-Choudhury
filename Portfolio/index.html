Bootcamp 2021: Interactive Sandpile

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). I created a sandpile animation where the user can decide what colours they can add or decrease to customise their sandpile to their fitting by using 'r', 'g', 'b' keys to increase the colour and the capital variants to decrease how much of the colours there are.

Check out the code on Github: https://github.com/YC-03/BootcampProject---Yusuf-Choudhury
